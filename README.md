📁 Folder Tree Viewer

フォルダを選択すると、最大階層までの構造をツリー形式で表示する Windows 用ツールです。
Python (tkinter) で作成したGUIアプリで、exe化して単体で動作させることもできます。

✨ 主な機能

📂 フォルダの階層構造をツリー表示

🔢 最大表示階層を指定可能

👀 隠しファイルを表示/非表示の切り替え

🧮 ファイル数カウント機能

📋 表示内容をクリップボードにコピー

💾 結果を tree.txt などのファイルに保存

🪟 完全ローカルで動作（インストール不要）

📦 ダウンロード

Windows実行ファイル（.exe）はリリースページからダウンロードできます。
